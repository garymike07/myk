Machine Learning API

About the Project

A scalable ML microservice for image recognition and natural language processing with auto-scaling capabilities.

Technologies Used

  • Python
  • TensorFlow
  • FastAPI
  • Docker
  • Kubernetes
  • AWS

Project Goals

The goal of this project was to create a highly scalable and reliable machine learning API that could be used to power a variety of applications. The API needed to be able to handle a large number of requests and provide accurate results in real-time.

Challenges and Solutions

One of the biggest challenges was to create a system that could automatically scale to meet demand. We solved this by using Kubernetes to manage our Docker containers and automatically scale them based on CPU and memory usage. We also used a load balancer to distribute traffic across our containers and ensure high availability.
